## Principle

Do not increase advertising spend on a message or channel until the results have been measured and the approach has been proven to work.

## Summary

Hopkins introduced advertising to the concept of split testing and coupon-based tracking. He argued that guessing which ad works is waste. The only reliable way to know what works is to test one variable at a time, measure the response, and scale only what is proven. Without tracking, there is no science, only opinion.

## Why it matters

This principle prevents teams from doubling down on weak messaging simply because it looks good internally. It forces marketing decisions to be based on data, not on which version the client or creative team prefers. It also protects budget from being spent on unproven approaches.

## How to apply it

1. Always define the success metric before launching any campaign, post or ad.
2. Test one variable at a time. Never change the headline, image, offer and audience in the same test.
3. Run tests long enough to collect meaningful data, but not so long that poor performance drains the budget.
4. Scale spend only on versions that measurably outperform the control.
5. Document what was tested, what the result was, and what was learned.

## Examples

- A split test between two headlines where only the headline changes and all other variables stay identical.
- A landing page test where one version leads with a specific offer and the other leads with brand messaging.
- A paid ads test that compares two audience segments using the same creative before scaling the better performing segment.

## Mistakes to avoid

- Drawing conclusions from sample sizes that are too small to be reliable.
- Changing multiple variables at once and claiming to know which one caused the result.
- Letting personal preference override test data.
- Stopping a test too early because one version appears to be winning before the data is statistically meaningful.
- Treating one successful test as universal truth without retesting in different contexts.
