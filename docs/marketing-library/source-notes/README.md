# Marketing Library Source Notes

Last updated: 2026-07-01
Status: Foundation

## Purpose

Source notes are the bridge between a book or research source and the Skill Cards that will later be extracted from it.

A source note exists to answer:

- What does this source contain?
- What can we reliably use it for?
- What can we not use it for?
- What would we need to verify manually?
- Which Skill Cards could come from this source?

## Ground rules

- Do not invent page numbers.
- Do not fabricate quotes.
- Do not pretend a source has been fully reviewed if it has not.
- These are starter templates and review plans, not final extracted research.
- A book on this list does not automatically produce Skill Cards. Each card must reference a specific principle, chapter or section.
- Review status must be honest about how much of the source has been covered.

## Review statuses

| Status | Meaning |
|---|---|
| Source identified, detailed extraction pending | The source is on the list. Structure exists but no detailed extraction has been done. |
| Partial extraction in progress | Some sections have been reviewed and preliminary notes exist. |
| Key principles extracted | Core chapters reviewed. Supporting sections may still need attention. |
| Extraction complete | Full source reviewed and all applicable Skill Cards drafted. |

## File naming convention

Use lowercase kebab-case: `title-of-source.md`

## What each source note should contain

- Source details
- Why this source matters
- What it can support
- What it cannot support
- Future Skill Card candidates
- Review status
- Notes to verify manually

## Relationship to Skill Cards

Source notes do not replace Skill Cards.

Skill Cards are extracted, structured knowledge units that agents can retrieve and apply.

Source notes are the planning and tracking layer that makes sure each Skill Card is honest about where it came from and how reliable it is.
