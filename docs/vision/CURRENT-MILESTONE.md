# Current Milestone

Last updated: 2026-07-01
Current milestone: Marketing Library foundation
Status: Active

## Milestone goal

Set up the foundation for the CG Dynamics Marketing Library and Skill Card system before building specialist AI agents.

The goal is not to make the AI generate more content yet. The goal is to create the trusted knowledge structure that future agents will use.

## Why this comes first

AI agents without a trusted knowledge base will produce generic AI output.

The Marketing Library must become the source of truth so agents can retrieve, reason and apply verified expertise.

## Scope for this milestone

### 1. Product documentation

- Create the master goal tracker.
- Create the current milestone tracker.
- Create the parking lot.
- Create the AI Workforce roadmap.
- Create the Marketing Library roadmap.

### 2. Knowledge structure

- Define the Skill Card schema.
- Define source types.
- Define confidence levels.
- Define industry tags.
- Define agent permission and usage rules.
- Define client specific knowledge boundaries.

### 3. App foundation

- Review the current CG Dynamics repository.
- Identify where a Marketing Library section should live.
- Identify existing client, staff and dashboard structures.
- Plan the database tables or storage model for Skill Cards.
- Plan the navigation separation between Client Intelligence, Operations Hub and AI Workforce.

### 4. First content batch

Create the first small batch of Skill Cards from trusted, non AI fluff sources.

Initial categories:

- Copywriting foundations.
- Buyer psychology.
- Advertising principles.
- Brand positioning.
- SEO fundamentals from official or trusted sources.
- Paid ads fundamentals from official platform documentation.

## Out of scope for this milestone

- Full AI agent execution.
- Automated poster generation.
- Client facing report redesign.
- Full task manager rebuild.
- CG Hours integration.
- Payroll related features.
- Large client specific libraries.

These can be added after the foundation exists.

## Definition of done

This milestone is complete when:

- The product vision documents exist in GitHub.
- The Marketing Library roadmap exists.
- The Skill Card template exists.
- The app has a clear planned place for the AI Workforce section.
- The first database or storage plan is agreed.
- The first trusted Skill Cards are ready to add or already added.

## Next milestone options

After this milestone, choose one:

1. Build the Marketing Library UI and storage.
2. Rework the app navigation into three divisions.
3. Improve Client Intelligence reporting quality.
4. Improve Operations Hub workflow usability.

Do not start all four at once.
