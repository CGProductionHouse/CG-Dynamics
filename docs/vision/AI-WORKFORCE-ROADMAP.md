# AI Workforce Roadmap

Last updated: 2026-07-01
Status: Draft roadmap

## Purpose

The AI Workforce is the specialist agent division inside CG Dynamics.

It should not be a generic chatbot area. It should become a structured workspace where AI agents use verified knowledge, active client context and internal learnings to support real business workflows.

## Core principle

AI must retrieve, reason and apply expertise.

AI must not invent expertise or produce generic content because it sounds good.

## Agent architecture

Every agent should have access to four knowledge layers where relevant:

1. Universal principles
2. South African market intelligence
3. Industry specific knowledge
4. Active client specific context

## Knowledge layers

### 1. Universal principles

Examples:

- Advertising principles.
- Copywriting frameworks.
- Buyer psychology.
- Consumer behaviour.
- Brand positioning.
- Offer design.
- Attention and memory.
- SEO fundamentals.
- Paid ads fundamentals.

Sources should prioritise established books, research papers, official documentation and verified data.

### 2. South African market intelligence

Examples:

- Platform usage.
- Search behaviour.
- Local trust signals.
- Regional and language differences.
- Cultural context.
- Seasonal and public holiday behaviour.
- WhatsApp usage.
- Local buying behaviour.

Sources should prioritise reputable reports, official statistics, platform data, research and internal campaign results.

### 3. Industry specific knowledge

Examples:

- Real estate.
- Restaurants and hospitality.
- Automotive.
- Construction.
- Architecture.
- Retail.
- Medical.
- Legal.
- Agriculture.
- Education.
- Tourism.

Industry libraries should define the audience, buying triggers, objections, search behaviour, content types, offers and conversion paths.

### 4. Active client specific context

Examples:

- Active client profile.
- Brand style.
- Tone.
- Audience.
- Services or products.
- Campaign history.
- Approved client notes.
- Client interviews.
- Current performance data.
- Internal account notes.

Client context must come from active client records and approved client knowledge. It must not be hardcoded into master roadmap documents.

## Initial specialist agents

### Marketing Strategist Agent

Purpose:
Turn marketing data, business context and Skill Cards into strategic recommendations.

Possible tasks:

- Campaign direction.
- Monthly recommendations.
- Market positioning notes.
- Offer improvement suggestions.
- Audience and message strategy.

### Copywriting Agent

Purpose:
Write captions, headlines, CTAs, website copy, ads and campaign messaging based on verified copywriting principles.

Must avoid:

- Generic AI wording.
- Empty benefit phrases.
- Vague hooks.
- Unsupported claims.

### Graphic Design First Draft Agent

Purpose:
Generate first draft design concepts or poster briefs using brand rules, content goal, hierarchy rules and marketing principles.

Important:
This agent should support the designer. It should not replace creative direction or final approval.

### Brand Guardian Agent

Purpose:
Review outputs before they leave the company.

Checks:

- Brand consistency.
- Generic AI wording.
- CTA clarity.
- Visual hierarchy.
- Audience fit.
- Message relevance.
- Evidence based reasoning.

### Paid Ads Agent

Purpose:
Support Meta and Google campaign planning, setup logic, performance interpretation and lead generation strategy.

Sources:

- Official platform documentation.
- Internal campaign data.
- Trusted paid media resources.
- Client performance history.

### SEO Agent

Purpose:
Support search visibility, content structure, keyword logic and search intent analysis.

Sources:

- Official search documentation.
- Reputable SEO research.
- Internal website performance data.
- Search query data where available.

### Client Report Agent

Purpose:
Prepare client facing report insights using performance data, campaign history, marketing principles and strategic recommendations.

Must answer:

- What happened?
- Why did it happen?
- Why does it matter?
- What should happen next?

### Creative Director Agent

Purpose:
Critique and improve work rather than generate from scratch.

Use cases:

- Poster review.
- Caption review.
- Campaign review.
- Brand consistency review.
- Content calendar review.

## Agent output rules

Every agent should, where possible, return:

- Recommendation.
- Reasoning.
- Source or Skill Card references.
- Confidence level.
- Risks.
- Next action.

## Anti generic language rule

Agents should flag vague words and phrases that make content feel AI generated.

Examples:

- Elevate.
- Unlock.
- Revolutionise.
- Seamless.
- Game changing.
- Cutting edge.
- Transform your business.
- Take it to the next level.

These words are not permanently banned in every context, but they should require justification.

## Implementation phases

### Phase 1 - Knowledge foundation

- Build Skill Card schema.
- Build Marketing Library structure.
- Add source management.
- Add confidence levels.
- Add tagging and relationships.

### Phase 2 - AI Workforce section

- Add navigation section.
- Add library views.
- Add Skill Card CRUD.
- Add source references.
- Add search and filters.

### Phase 3 - First agents

- Start with critique and planning agents before full generation agents.
- Brand Guardian and Copywriting Agent are strong candidates for first build.

### Phase 4 - Plug into Client Intelligence

- Use agents to explain data.
- Generate report recommendations.
- Improve client dashboard insights.

### Phase 5 - Plug into Operations Hub

- Use agents to help with briefs, task creation, content planning and production QA.

## Long term goal

The AI Workforce should become a trained internal department that understands marketing, industry context, local audience behaviour and active client needs.

The value is not that AI can write faster. The value is that the company can apply better thinking more consistently.
